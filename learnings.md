# Learnings from Active Directory User Management Lab1

These are essential identity and account‑management tasks used daily in real IT environments. These actions help maintain security, support end users, and ensure proper access control across the organization.

---

## 🔐 1. Resetting a User Password

- Locate the user account inside the correct Organizational Unit.
- Right‑click the account and select **Reset Password**.
- Enter a new temporary password and choose whether the user must change it at next logon.
- Confirm the reset and communicate the new credentials securely.

---

## 🔓 2. Unlocking a Locked Account

- Open the user’s properties in ADUC.
- Go to the **Account** tab.
- Check the box labeled **Unlock account** (only visible when the account is locked).
- Apply the change and verify the user can sign in again.

---

## 🚫 3. Disabling an Account (Employee Offboarding)

- Locate the user account in ADUC.
- Right‑click → **Disable Account**.
- Confirm the action (the icon changes to a downward arrow).
- Move the account to a designated **Disabled Users** OU for organization and policy control.
