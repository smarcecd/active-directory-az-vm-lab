# Active Directory Lab — Project Overview

This project demonstrates the deployment and configuration of an Active Directory Domain Services (AD DS) environment using a Windows Server virtual machine hosted in Microsoft Azure.  
The goal of this lab is to build hands-on experience with identity management, domain administration, automation, and common help desk operations using both the GUI and PowerShell.

---

## 🎯 Purpose of the Lab

The lab was designed to help me:

- Understand how Active Directory works in a real environment  
- Practice deploying a domain controller from scratch  
- Automate AD tasks using PowerShell  
- Build and manage Organizational Units, users, groups, and policies  
- Perform common IT support tasks such as password resets and account unlocks  
- Strengthen my cloud and infrastructure skills using Azure  

---

## 🧱 What I Built

This lab includes:

- A Windows Server 2025 VM deployed in Azure  
- A new AD forest and domain: **lab.local**  
- Organizational Units for IT, Finance, HR, Sales, and Computers  
- Security groups for each department  
- User accounts created and assigned to their respective groups  
- Group Policy configuration (password policies, security settings, etc.)  
- PowerShell automation for deployment and management  

---
